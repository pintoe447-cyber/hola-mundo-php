# Hola Mundo en PHP

## Descripción
Este proyecto consiste en una aplicación básica desarrollada en PHP que muestra en pantalla el mensaje “Hola mundo, soy Edwin Pinto” e incluye una imagen creada con inteligencia artificial.

## Prerrequisitos
- Tener instalado PHP
- Tener acceso a una terminal, navegador o servidor local
- Un editor de código como Visual Studio Code

## Pasos para su ejecución
1. Descargar o abrir los archivos del proyecto.
2. Verificar que PHP esté instalado en el equipo.
3. Ejecutar el archivo desde la terminal o en un servidor local.
4. Visualizar en pantalla el mensaje y la imagen.

## Ejemplo de ejecución
php index.php
